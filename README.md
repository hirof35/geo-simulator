# 2D Geometry Simulator

TypeScript と HTML5 Canvas で実装された、リアルタイム 2D 幾何学シミュレーターです。
画面上の点をマウスでドラッグすることで、「線分」と「円」の交点座標が数学的にどう変化するかを視覚的に観察できます。

# 開発サーバーの起動

起動後、ブラウザで http://localhost:5173 にアクセスしてください。🎮 使い方画面上の4つの点をマウスでドラッグして動かすことができます。点 A / 点 B: 線分の端点です。点 C: 円の中心点です（赤色）。点 D: 円周上の点です。CからDの距離がそのまま円の「半径」になります。緑色の点: 線分と円の交点です。交点が存在するときだけ自動的に計算されて表示されます。📝 技術スタック言語: TypeScript描画: HTML5 Canvas API (2D Context)ビルドツール: Vite📐 数学的背景（交点判定の仕組み）このシミュレーターは、線分のベクトル方程式と円の方程式を連立させた 二次方程式の判別式 を用いて交点をリアルタイムに計算しています。線分のベクトル方程式:$P(t) = A + t(B - A)$  （ただし $0 \le t \le 1$）円の方程式:$\|P - C\|^2 = r^2$判定方法:上記を連立させてできる二次方程式 $at^2 + bt + c = 0$ の判別式 $D = b^2 - 4ac$ を評価します。$D < 0$: 交点なし$D \ge 0$: 解 $t$ を求め、 $0 \le t \le 1$ の範囲にあるもののみを有効な交点（線分上の点）として描画します。
